# 2048游戏

一个功能丰富、视觉精美的2048游戏实现，具有现代化的UI设计和完善的游戏体验。

## 功能特性

### 核心游戏功能
- ✨ 完整的2048游戏逻辑和滑动合并机制
- 🎯 实时得分计算和最高分持久化
- 📱 响应式设计，支持PC端和手机端
- 🖱️ 键盘控制（PC端）和触摸滑动（手机端）
- 🎮 开始游戏和暂停/继续功能

### 视觉设计
- 🎨 B站蓝色科技风格，带有霓虹光效过渡
- 🌧️ 黑客帝国代码雨背景动画
- 🎭 主题切换功能（科技风格/经典风格）
- 💫 数字块移动轨迹和合并动画
- 📺 模拟Three.js技术水印效果

### 用户体验
- 📊 中英文语言切换
- ⚙️ PC/手机端模式切换
- 📈 得分和最高分统计
- 🎪 加载页动画效果
- 🔄 新游戏和重新开始功能

## 安装和运行

1. **克隆或下载项目文件**
   将项目文件下载到本地目录。

2. **启动本地服务器**
   由于游戏使用了本地存储和JavaScript模块，建议通过本地服务器运行：

   ```bash
   # 使用Python 3
   python -m http.server 8080

   # 或使用Python 2
   python -m SimpleHTTPServer 8080

   # 或使用Node.js的http-server
   npx http-server -p 8080
   ```

3. **访问游戏**
   在浏览器中打开 `http://localhost:8080` 即可开始游戏。

## 游戏操作

### PC端
- 使用**方向键**（↑ ↓ ← →）移动数字块
- 点击**New Game**开始新游戏
- 点击**暂停**按钮暂停游戏

### 手机端
- **滑动手指**在屏幕上移动数字块
- 点击**开始游戏**按钮开始游戏
- 点击**暂停**按钮暂停游戏

## 游戏规则

1. 使用方向键或滑动屏幕移动所有数字块
2. 相同数字的方块在碰撞时会合并成一个更大的数字
3. 每次移动后，随机位置会生成一个2或4的数字块
4. 当达到2048数字块时获胜
5. 当无法移动任何数字块时游戏结束

## 技术实现

### 核心技术栈
- **HTML5**：游戏结构和布局
- **CSS3**：样式设计、动画效果和响应式布局
- **JavaScript (ES6+)**：游戏逻辑、交互和数据管理
- **localStorage**：用户偏好和最高分持久化
