# 2048 Game

一个功能丰富、视觉效果精美的2048游戏实现，具有科技风格和经典风格两种主题，支持PC和手机模式切换。

## 功能特性

### 核心游戏玩法
- 经典的2048游戏机制：使用方向键移动数字方块，相同数字的方块碰撞时会合并
- 得分计算和最高分记录（使用localStorage持久化）
- 游戏结束检测

### 视觉效果
- **科技风格**：B站蓝色科技风，霓虹光效过渡，黑客帝国代码雨背景
- **经典风格**：还原原版2048的视觉风格
- **数字块动画**：平滑的移动轨迹和合并动画
- **加载页**：带有"@江鱼"标识的加载页面，包含脉冲动画和进度条
- **技术水印**：模拟Three.js渲染效果的水印，包含"@江鱼"标识

### 响应式设计
- **自动模式**：根据屏幕尺寸自动切换PC/手机布局
- **手动模式**：可以手动选择PC或手机模式
- **移动优化**：针对手机屏幕优化的布局和触摸支持

### 其他特性
- 主题切换功能（科技/经典）
- 游戏状态持久化
- 平滑的动画效果
- 视觉反馈丰富

## 安装和运行

### 本地运行
1. 克隆或下载项目文件
2. 直接在浏览器中打开`index.html`文件
3. 开始游戏！

### 部署到服务器
可以将项目文件部署到任何静态文件服务器上，如GitHub Pages、Netlify等。

## 如何游玩

1. 使用**方向键**（↑↓←→）移动数字方块
2. 相同数字的方块在移动过程中碰撞会合并
3. 每次移动后，会在空白位置随机生成一个2或4的数字块
4. 目标是合并出2048数字块
5. 当无法移动时，游戏结束

## 技术栈

- **HTML5**：页面结构
- **CSS3**：样式和动画效果
  - CSS变量
  - 过渡和动画
  - 响应式设计
  - 3D变换
- **JavaScript**：游戏逻辑和交互
  - ES6+ 类和模块化
  - Canvas API（用于代码雨背景）
  - localStorage（用于数据持久化）

## 主要文件说明

### index.html
包含游戏的基本HTML结构，包括加载页、游戏容器、分数面板、控制按钮等。

### style.css
定义了游戏的所有样式，包括：
- 主题样式（科技/经典）
- 动画效果（加载动画、移动动画、合并动画）
- 响应式布局
- 模式样式（PC/手机）

### script.js
实现了游戏的核心逻辑：
- Game2048类：处理游戏的核心逻辑
- ThemeManager类：管理主题和模式切换
- MatrixRain类：实现代码雨背景动画
- 游戏状态管理和持久化
